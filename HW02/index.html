The variables to be compared can be selected using X axis and Y axis dropdowns. Scatter Plots

Scatterplot is a mathematical graph/diagram which is plotted on a catesian plot, and is used to compare two variables plotted on 2 axes of cartesian plane. Scatteplot are extremely useful to find correlations whether the correlation is positive, negative or uncorrelated. Scatterplots are also useful to find the spread of data and finding outliers in the dataset. One can add other variables by adding visual cues likes color, shape, size etc on a 2D surface. Scatterplots can be likewise three dimensional as well. Scatterplots come with great amount of variation. Scatterplot feature bring thi variation. Examples of these features include color, size, opacity, zoom level, shape, different ticks, color hues, different scales in axes (e.g. linear scale, logarithmic scale), normalization, standardization, multiple sub scatter plots within one bigger encompassing plot, using different width and height of subplot, different stroke color, scatterplot with a specific image in background.

A novel scatterplot example could be a 3D scatterplot. This would enable us to find correlation between 3 variables and with additional visual cue we could add more variable information to the 3D space of scatterplot.
